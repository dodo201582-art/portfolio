Project Summary

카이스트(KAIST)의 학사 서비스인 LMS(Learning Management System)를 위한 인프라 아키텍처를 재설계하고 구축한 프로젝트입니다. 안정적인 학습 환경 제공을 목표로 성능 저하 원인을 분석하고 최적화하였으며, CDN과 Load Balancer를 도입하여 콘텐츠 전송 속도를 획기적으로 개선하였습니다.

Tech Stack

NHN Cloud CDN Load Balancer Web Server WAS Linux Performance Tuning

Key Achievements

인프라 안정화: 안정적인 학습 환경 제공을 위한 인프라 재설계 및 구축 완료

성능 최적화: 서버 내 부하 프로세스 점검 및 최적화를 통해 성능 저하 원인 분석 및 해결

운영 효율성 증대: CDN 및 Load Balancer 적용으로 콘텐츠 전송 속도 개선 및 원활한 LMS 운영 확보
