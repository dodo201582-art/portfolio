Project Summary

NIA(한국지능정보사회진흥원) 주관 민간 클라우드 마이그레이션 사업으로, 기존 온프레미스 환경을 NHN 공공 클라우드로 이전한 프로젝트입니다. 단순 이전을 넘어 고가용성(HA) 아키텍처로 전면 재설계하여 서비스 안정성을 확보하고, 백업 및 보안 체계를 강화하여 공공 서비스의 신뢰성을 높였습니다.

Tech Stack

NHN Cloud Apache Tomcat Session Clustering Acronis Backup CentOS

Key Achievements

서비스 가용성 극대화: WAS 및 DB 이중화(Active-Standby) 구성 및 Tomcat Session Clustering 적용으로 장애 발생 시 중단 없는 서비스 환경 구축

데이터 보호 체계 확립: Acronis Backup 솔루션 도입 및 최적화된 백업 정책 수립으로 데이터 손실 리스크 제거

보안성 강화: Apache 웹 서버 설정 최적화 및 보안 취약점 조치로 안전한 웹 서비스 운영 환경 마련
