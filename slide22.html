Project Summary

기존 On-premise 환경을 KT Cloud로 이전하며 WebtoB 및 JEUS 미들웨어의 버전 업그레이드를 동시에 수행한 마이그레이션 프로젝트입니다. IPSec VPN을 통해 기존 내부 Oracle DB와의 하이브리드 연동을 구현하였으며, NAT Gateway 미지원 환경을 극복하기 위해 HAProxy를 활용하여 안전하고 효율적인 외부 통신 아키텍처를 설계했습니다.

Tech Stack

KT Cloud IPSec VPN HAProxy WebtoB JEUS Oracle DB (Hybrid) rsync Load Balancer

Key Achievements

성공적인 클라우드 전환: 안정적인 데이터 이관 및 서비스 성능/운영 효율성 향상

고가용성 확보: Load Balancer 및 JEUS 컨테이너 기반 서비스 이중화 구성 완료

하이브리드 연동 구현: IPSec VPN을 활용하여 클라우드 WEB과 On-premise DB 간 통신 설계

외부 연동 아키텍처: HAProxy를 통해 CDN, FTP, 전자결제 등 외부 서비스와의 원활한 연동 지원
