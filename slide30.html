Project Summary

NAVER Cloud Platform(NCP) 기반의 빅데이터 플랫폼 인프라 아키텍처 설계 및 구축 프로젝트입니다. 공공 클라우드 환경에서의 안정성, 보안성, 확장성을 최우선으로 고려하여 네트워크 및 보안 아키텍처를 수립하였으며, 엄격한 보안 규정을 준수하는 표준 운영 환경을 구축하였습니다.

Tech Stack
NAVER Cloud Platform VPC / Subnet ACG / NACL WAF IPSec VPN Linux Security Monitoring
Key Achievements

보안 네트워크 아키텍처 설계: Zone 분리, ACL 정책, 라우팅 설계를 통해 외부 위협으로부터 안전한 데이터 처리 환경 구축

인프라 표준화 및 운영 최적화: 서비스 가용성과 확장성을 고려한 인프라 구성 표준 수립 및 운영 매뉴얼화

보안 취약점 조치 완수: OS 및 애플리케이션 계층 취약점 점검과 선제적 조치로 높은 수준의 보안성 확보
